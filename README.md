# vLLM Paper Reproduction (CSE 231)

Reconstruction of Kwon et al., *Efficient Memory Management for Large Language Model Serving with PagedAttention* (SOSP '23) on a single Kaggle T4.

Open `run.ipynb` and run the cells in order. Restart the kernel between vLLM-based cells (a single T4 cannot hold two vLLM engines at once), then re-run setup and helpers after each restart.
